Acerca de Dataworld

Desde que en marzo del 2020 la epidemia de Covid-19 se hiciera global, estar informado de lo que sucede en el momento empezó a ser una necsidad, como sociedad nos dimos cuenta de que es importante conocer el estado actual del mundo.

Dataworld es una aplicación web que recopila información importante y concisa de los temas principales que deben ser conocidos. Todo con base en fuentes confiables de información:

  • Clima y pronósticos: API Clima
  • Información actual del Covid-19: API Covid
  • Últimas noticas referentes a ciencia, tecnología y salud: API Noticias

Un proyecto de DataMind

Dataworld es la primera aplicación desarrollada por DataMind.

DataMind es una empresa dedicada a brindar soluciones digitales mediante software de calidad, para la mejora de procesos y toma de decisiones, proporcionando una operación más segura, productiva y medible para los clientes.
